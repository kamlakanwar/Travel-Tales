## Setup and Installation

<p>To run this project locally, follow these steps:</p>
 
1. **Clone the repository:**
    ```bash
    git clone https://github.com/kamlakanwar/Travel-Tales
    ```
2. **Navigate to the project directory:**
    ```bash
    cd Travel-Tales
    ```
